# Udacity Neighborhood Map Project
This project is for Udacity Front-end Developer course. It's main purpose is to demonstrate MVC architecture and AJAX request.

# How to run
- Open Folder Neighbourhood Map.
- Open File index.html.
- You will be able to see a map and markers on it.
- Click on the markers to get ratings and likes for selected location.
- Location can also be filtered using the seachbox provided.

# How it works
- Google api to load map and markers.
- foursquare api to fetch data.
- use knockout to bind data.
- open app.js to open to see java script.
